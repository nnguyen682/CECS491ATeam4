Good Vibes Summarized

Good Vibes is an Amazon Alexa skill that aims to assist users with depression.
Although Alexa may not be physically present, her interactions will be conversational
in the form of a voice user interface.

Why did we create Good Vibes?

According to TechCrunch, 16 percent of Americans own a smartspeaker.(1) More specifically,
GeekWire states that there are more than 8 million people that own an Alexa Device.(2) Healthline
stated in an article that there is a recorded amount of 16 million cases of people that experienced
at least one major depressive episode in 2012.(3)

Depression takes on many forms and is often difficult to diagnose. It can be common that somebody
in a current state of depression does not have the desire to seek help, let alone leave their house.
This, is where our application has the benefit of easy in-home access for anybody with an Echo device.

This web page will show you how to contribute to the development of the Good Vibes Alexa skill.

Main Features

Assessment Tests

Taking assessment tests will allow the skill to have a better understanding of the user’s state of
depression. This will in turn allow the skill to respond with articles and more personalized responses
to try to help the user.

Express Feelings

Users will be able to talk about how they are feeling to Good Vibes and the skill will try to provide useful
responses. If the user states that they are having a nice day, then Good Vibes may respond with positive reinforcement
such as “Great moves, keep it up, proud of you”. If the user states that they are feeling upset, then Good Vibes might
give the user a suggestion such as “ You deserve some relaxation. How about watching a movie to take your mind off of things?”
or Good Vibes might try to give other forms of support like, “I am here for you and I am not going anywhere.”

Trigger Words

Since depression is a serious topic that cannot be taken lightly, we have to consider some extreme cases. This might include
suicidal or self-harmful thoughts, that Good Vibes will try to hear out. If an event like this were to occur, Good Vibes will
suggest seeking out help while also reassuring that the user still has a purpose in life. Good Vibes will also provide the number
for the suicide hotline.

Contribute

If you would like to contribute to the development of Good Vibes you can access the source code here at the Github repository
Feel free to inform us of any bugs you come across, fork the project, or download the source code.

There will be three files in the Github repository for this project.
README.md.txt contains an overview for the project goals and intentions.
index.js is the file that is used to handle Alexa responses through AWS Lambda.
json is the file required for Alexa to analyze user intents, slot values, and utterances.

Software Used for Development

AWS Lamda is a computing platform that is part of the Amazon Web Services. Lambda is used to develop the back end of the skill.
The computing service of AWS lambda is event-driven allows developers to write code for events or interactions with Alexa.

Alexa developer portal is used to develop the front end of a skill. Inside the portal is where developers are able to add intents, slots, and utterances.
The developer portal also allows developers to connect their skill to their AWS lambda function.

node.js is an open-source JavaScript runtime environment that executes JavaScript code on its server-side. Alexa developers primarily use node.js to develop Alexa
Skills because of a package that includes the Alexa Skill Development kit. Good Vibes uses the Alexa SDK for various functionalities and access to development tools.

DynamoDB is a NoSQL database service that AWS offers to allow fast and scalable storage of user information. We use DynamoDB with Good Vibes
to store user preferences and assessment test results. AWS Lambda interacts with the database table to give user-specific outputs.


Frequently Asked Questions (FAQ)

Q: Why did the team at Good vibes choose node.js instead of other languages such as Python or Java?
     A: The team at Good Vibes chose to develop with node.js because of the Alexa SDK that enables developers to focus on creating engaging features rather than writing hundreds of lines of code. Other languages are great for different purposes, such as Python with machine learning, but node.js provides the easiest way for developers to create Alexa skills.

Q: What languages do I need to know to develop a skill like Good Vibes?
     A: Alexa skills can be written in almost any programming language. You can simply choose the language that best fits your skill set and knowledge; however, the team at Good Vibes recommends starting off with node.js because of the Alexa Skill Development Kit and other tools that can accessed through node.js.

Q: Do I need experience with programming to get started developing Alexa Skills?
     A: No you do not need to be an expert at programming to develop skills. When you are creating a skill, you can select an option to start with a blueprint that contains the code needed in the AWS lambda function. You can always edit the code in a blueprint or start from scratch if you have prior knowledge of programming.

Q: Why should I choose Good Vibes instead of using traditional measures to aid depression?
     A: Good Vibes provides a human-like interaction that is both free and convenient for users. Alexa is a great alternative for those that do not want to talk to anyone about their depression. Users will not have to live with the fear of judgment using Good Vibes since Alexa is here with the sole purpose of providing assistance.
